# The Complete ML / DL / NLP / LLM Playbook

A from-zero-to-domain-expert curriculum. Every idea is shown **three ways**: plain words → a tiny
worked example **by hand** → code that **re-computes the same numbers** to verify it. Runnable cells
use `numpy / pandas / scikit-learn / torch / matplotlib`; TensorFlow & HuggingFace appear as
clearly-marked **read-only** cells (blocked by enterprise security on this machine).

Work through them in order. At **P20** the playbook hands off to the existing **TinyGPT notebooks**
(`../notebooks/00`–`13`); after `notebooks/13`, return here for Part 5. Full plan: `../PLAYBOOK_ROADMAP.md`.
